# Support Ticket System

A web-based Support Ticket System developed using the **PHP Laravel framework**. This application serves as a technical support platform where clients can submit issues, and administrators can manage and resolve them.

## Features
- **User Authentication & Registration:** Secure sign-up, sign-in, and sign-out functionality.
- **Role-Based Access Control:** Differentiated dashboards and features for **Clients** and **Administrators**.
- **Client Dashboard:** Form to submit technical issues with custom titles, descriptions, categories, and priorities. Real-time list showing submitted tickets and their status.
- **Admin Panel:** Ability for administrators to view all submitted tickets and mark them as "Resolved".
- **Database Architecture:** Structured relational MySQL database ensuring data integrity.

## Prerequisites
- PHP >= 8.2
- Composer
- XAMPP / MySQL Server
